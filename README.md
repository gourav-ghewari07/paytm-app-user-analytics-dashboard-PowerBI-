💳 Paytm App User Analytics Dashboard

📌 Project Overview

The Paytm App User Analytics Dashboard is an interactive Power BI project developed to analyze user behavior, transaction performance, customer retention, and feature adoption within a digital payment ecosystem.

This dashboard provides a comprehensive view of transaction trends, user engagement, transaction success rates, and platform usage patterns, enabling data-driven business decisions.

---

🎯 Business Objective

The objective of this project is to:

- Analyze transaction performance.
- Monitor customer retention and churn.
- Evaluate transaction success rates.
- Understand feature usage patterns.
- Identify user behavior across different device types and city tiers.
- Support strategic business decisions through analytics.

🔍 Key Insights

1. Strong Transaction Volume

The platform processed approximately 188K transactions with a total value of 32M, demonstrating significant user activity.

2. High Transaction Success Rate

A success rate of 92.44% indicates reliable transaction processing and positive customer experience.

3. Android Dominates Platform Usage

Most users access the platform through Android devices, highlighting the importance of Android-focused optimization.

4. Feature Usage Is Well Distributed

Users actively utilize multiple services including UPI payments, bill payments, recharges, wallets, loans, and insurance products.

5. Declining Monthly Active Users

Monthly active user trends show a gradual decline in later months, suggesting potential customer engagement challenges.

6. Transaction Volume Dropped in 2024

Transaction count decreased compared to previous years, indicating a need for customer retention and growth strategies.

7. City Tier Analysis Reveals Market Opportunities

Different city tiers contribute differently to platform usage, helping identify expansion opportunities.
